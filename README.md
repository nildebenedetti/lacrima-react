# LACRIMA Frontend

LACRIMA is an imaginary skincare e-commerce created as a personal project to combine two worlds I genuinely enjoy: software development and brand creation.

The goal was to build my first complete React e-commerce while strengthening the core technologies covered during my Full Stack Master's program, with a particular focus on JavaScript, React and modern frontend development.

Beyond the technical side, this project gave me the opportunity to reconnect with my previous experience in digital marketing. From the brand identity and UX to product design, copywriting and visual storytelling, every element was designed from scratch. Without budget constraints or commercial limitations, I simply allowed creativity to lead the way.

LACRIMA is also a small tribute to the gentle souls who still choose kindness, sensitivity and imagination. To those who believe that fantasy is not an escape from reality, but another beautiful way to experience it.

## Main goals

- Build a complete React e-commerce application
- Strengthen React, JavaScript and frontend architecture skills
- Design a coherent user experience from branding to checkout
- Experiment with component-based design and state management
- Create a fictional luxury skincare brand with a complete visual identity
